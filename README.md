# BoatLog — Boat Maintenance Checklist (pilot)

One QR sticker on the boat. Scan it and the right checklist comes up — daily pre-departure and post-trip, weekly/monthly upkeep, and engine-hour services that trigger themselves as you log hours.

**Live app:** https://parker0125-pixel.github.io/boatlog/

Pilot build for use-case feedback. All data stays on the phone that scans (nothing is uploaded).
